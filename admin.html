Las respuestas del formulario se guardarán en tu Google Sheet.
¿Cómo configurar Google Sheets?

1. Creá una Google Sheet nueva para este evento.

2. En la hoja, ir a Extensiones → Apps Script.

3. Reemplazá todo el código con esto:

function doPost(e){ var ss=SpreadsheetApp.getActiveSpreadsheet(); var sh=ss.getActiveSheet(); if(sh.getLastRow()===0) sh.appendRow(['Fecha','Nombre','Apellido','Asistencia','Acompañantes','Restricciones','Mensaje','Evento']); var d=JSON.parse(e.postData.contents); sh.appendRow([new Date().toLocaleString('es-AR'), d.nombre||'',d.apellido||'',d.asistencia||'', d.acompanante||'',d.restricciones||'',d.mensaje||'',d.evento||'']); return ContentService.createTextOutput(JSON.stringify({ok:true})) .setMimeType(ContentService.MimeType.JSON); }

4. Guardar → Implementar → Nueva implementación.

5. Tipo: Aplicación web · Ejecutar como: Yo · Acceso: Cualquier persona.

6. Copiá la URL del deploy y pegala arriba.

Si lo completás, también recibirás un email por cada RSVP.

💡 Consejos para el video
